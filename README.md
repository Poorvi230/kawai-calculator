# 🎀 Kawaii Calculator - Hello Kitty Edition

A delightfully cute and interactive calculator themed around Hello Kitty with a "rage" mechanic, stunning visuals, and mobile-friendly design!

## ✨ Features

### Core Calculator Functionality
- **Full arithmetic operations** - Addition, subtraction, multiplication, division, and modulo
- **Secure math evaluation** - Uses safe parsing to prevent code injection attacks
- **Real-time display** - Shows calculations as you type
- **Clear and backspace** - Easy input correction

### Kawaii Features 🐾
- **Rage Meter System** - Fills up as you use the calculator, unlocking chaotic behavior at higher levels
- **Annoyed Hello Kitty Animation** - Shows adorable 😾 kitty throwing a tantrum with "FAHHH" text before revealing answers
- **Intelligent Character Swaps** - At higher rage levels, occasionally swaps numbers with cute letters (like 0→O, 1→l)
- **Random Double Backspace** - Higher rage levels cause unpredictable deletions
- **Popup Interruptions** - Hello Kitty randomly interrupts calculations with funny system messages

### Visual Enhancements
- **Interactive Background** - Particles, trails, and sparkles respond to mouse movement and clicks
- **Floating Hello Kitty SVGs** - High-quality SVG characters drift gracefully across the page
- **Multiple Animation Types** - Bows, flowers, sparkles, and stars with unique floating patterns
- **Click Bursts** - Click the background to create particle explosions
- **Mouse Tracking** - Magical trails follow your cursor

### Mobile Friendly 📱
- **Fully Responsive Design** - Adapts beautifully to phones, tablets, and desktops
- **Touch-Optimized Buttons** - Large touch targets for easy interaction
- **Optimized Visuals** - Lighter particle effects on mobile for better performance

## 🎮 How to Play

1. **Click buttons** to enter numbers and operators
2. **Press "="** to calculate (watch for the adorable kitty tantrum!)
3. **Click the title** to reset the rage meter (cheater detected 😏)
4. **Move your mouse** around for interactive particle trails
5. **Click the background** to create particle bursts
6. **Use the calculator repeatedly** to increase rage level (every 3 calculations)

### Rage Level Effects
- **Levels 1-3** - Occasional number swaps and false errors
- **Levels 4-6** - More frequent character replacements and double backspaces
- **Levels 7-10** - Maximum chaos! Frequent interruptions and rage popups

## 🔒 Security

- ✅ No unsafe `eval()` - Uses validated math parsing with strict whitelist
- ✅ Input validation - Only allows digits, operators, and parentheses
- ✅ XSS-safe - All user input is sanitized
- ✅ No external dependencies - Pure vanilla JavaScript

## 🚀 Getting Started

### Local Development
1. Clone this repository
2. Open `index.html` in your web browser
3. Or run a local server:
   ```powershell
   cd 'path\to\Kawai Calculator'
   python -m http.server 8000
   ```
4. Visit `http://localhost:8000`

### Hosting
This is a single HTML file with no backend requirements. Deploy to:
- **GitHub Pages** - Free hosting for static sites
- **Vercel** - Zero-config deployment
- **Netlify** - Drag-and-drop deployment
- **Any static web host** - Works anywhere!
